Clean Dictionary
============

Python implemention of a **No Ad** Dictionary

## Dependency
* Python3
* PyQt5
* Pyinstaller*

## Environment
* Windows or MacOS

## Usage

```bash
python CleanDict.pyw
```

## Package to executable

setup requirements

```bash
pip install --upgrade PyInstaller pyinstaller-hooks-contrib Pillow
```

packaging

```bash
pip
pyinstaller  --windowed --icon=dict.ico CleanDict.pyw 
```
